也就在半个月前，我觉得做网站是件很难的事情，好像对于自己而言那是遥不可及的事情。

现在，能在自己亲手打造的博客里写文章的我再次想想做网站的事，想扔下一句话：也就是那么回事……（不过，做网站是一会儿事，做好就是另一会儿事了，想做好一个网站真的是很不容易）

是的，我想这就是隔行如隔山的道理吧，事实上很多事情根本没有你想想的那么难，往往人们缺少的是一颗敢闯敢干的心而已。

闲话不多说了，正式进入主题。在我看来，对于初学者来说建站的最大的困惑是不了解网站工作的原理，这就像一个黑匣子，你不知道里面装得什么，所以觉得它难它神秘。解决这个问题的方法很简单，把匣子打开看看就行了，打开以后你会发现原来盒子里的东西根本没那么复杂，一共就三样东西：域名，空间，网页。

简单做下解释：

1. 域名: 域名就是网站的地址，域名和网站是一一对应的，没有重名的。例如：www.baidu.com就是百度的域名，dreamhunter.me就是我的小站的域名等等。域名的作用就是为了人们记忆方便，如果没有域名的话你需要记住一长串的ip地址，如百度的ip是123.125.114.144,你在浏览器的地址栏里面输入ip地址和输入baidu.com效果是一样的，但是人类是不善与记数字的，于是发明了域名。也许有人第一次见.me的域名，觉得很神奇，这主要是因为大多数的网站都用.com的域名，其他种类的域名相对见的少的原因而已。域名的后缀名有很多种，分为Generic Top Level Domain (gTLD) Extensions和Country Code Top Level Domain (ccTLD) Extensions两大类，如我的域名后缀是.me，这是 Montenegro（黑山）的域名后缀，就像.cn代表中国一样。其实现在这些概念都不重要，重要的是你喜欢，因为这两类域名的效力上是没有什么区别的。选域名是网建的第一步，一定要想一个自己喜欢的，这很重要。我已经够罗嗦了，其实这方面的资料网上很多，还有不懂的去搜一下就好了。给你个域名注册商的网站，去逛逛吧：http://www.name.com。 页面最下面有域名的种类和价格。
2. 空间：空间就是储存你网站的地方。你用你的电脑的时候各种文件是储存在硬盘里的，而你的网站文件是储存在你的空间里的。你要是做中小型网站或个人博客的话去租人家的空间好了，也就是通常所说的共享空间(shared hosting)。你需要每个月或者每年给你的空间商一笔租赁费，费用问题最后一起说，这里略过。如果你要做大型网站的话就要自己买服务器了，我不解释了，估计看着篇文章的人99.9%的是新手级别的。
3. 网页：买好域名了，也买好空间了，那么剩下的事情就简单了，把你的网页放到你的空间里去，然后把你的域名“指”到你的空间上去就完了。这样别人输入你的域名，域名就指向你的空间，然后空间把你的网站拿给别人看。网页制作方面的问题我不敢多谈，因为我现在也只是局限于使用WordPress做博客而已，但道理都是一样的，至于你用什么软件制作你的网页那就是萝卜青菜了，最终都是要传到空间上去给别人看的。如今其实你不许要掌握任何知识都可以作出非常漂亮的网站，因为有海量的模板可以供你选择，海量的插件供你挑，你只需要点鼠标而已。

以上三条只是一个铺垫而已，你可能已经在无数的地方看到过这方面的介绍，那么下面给您看点真货，有些是我以前遇到过的问题，有些是经验之谈，希望对您有帮助：）

* 域名注册的相关问题

  1. 域名注册商的选择？建议在美国注册域名，原因很多了，比国内便宜，比国内服务好权限大，不用备案……在google上搜一下“domain name registration”，会出来一片域名注册商。你要是有功夫的话可以一个一个看，值得说明的是大多数的域名注册上也提供空间服务，而且经常会搞活动，比如买空间送域名啊什么的。如果你想省事的话可以在一家即卖域名又卖空间的商家那里一并买了。 最近比较火热的域名商是Go Daddy（总让我联想到 Lady GaGa~ ），但我本人对他家的印象不好，因为他每天都会给我发垃圾邮件，所以你在他家注册会员的时候一定要做好每天收3～4封垃圾邮件的心理准备……，其实最终没选择它的主要原因是他家的东西性价比不是最高的。我最终选择了在 name.com 注册我的域名（尽管价格不是最低的，但感觉服务比较好），name.com是只铁公鸡，很少打折（不像Go Daddy天天告诉你他家东西打折啦，快来买啊～），但是服务上是绝对可以信得过的，而且送免费的Private（用Coupon,后面会说。不知到Private是什么的话也自己Google下吧，不细说了）。另外有几点需要注意的是，所有的商家广告上说的 Free Domain 都是建立在购买他家特定的空间的基础上的。所有的打折的域名只有在你购买一年的情况下才有效，例如name.com上说我们家的.me域名搞活动，第一年7块9毛9分钱。结果你一高兴买了2年，你会发现两年的价格是37块9毛8分，这是因为你购买了两年的话第一年的就不打折了！所以建议看见打折的域名可以先买一年，然后再 renew，这样比较省钱：）
  2. 后缀不同的域名的功能相同么？据我了解是完全相同的，前面说过这只是个地址而已，它只负责指向你的空间。还有一些域名有特殊的含义，是不能随便注册的。

* 主机空间问题

  1. 主机是 linux 还是 windows 有啥区别？当你买空间的时候要选择租 linux 的主机还是 windows 的主机，主要区别就是它们提供的应用程序有所不同。另外，linux 会相对便宜一些。
  2. 带宽是啥？简单的说就是流量，每个月给你的空间的流量限制。一般对于个人网站可以不考虑它，因为一般是不可能达那么大的带宽的。
  3. 空间大小是个什么概念？你要是做视频或图片网站的话可能需要大一点的空间，如果是单纯写文章的话1G空间够你写很久很久了，可以自己算一下1G的大小是多少汉字……
  4. 空间主机推荐。我只推荐我现在用的主机商Arvixe，网址 www.arvixe.com，我觉得他们家的东西真的挺不错的，性价比极高（可以用打折的coupon），你可以去他们主页上看一眼。最重要的一点是：服务好！真的，我从关注他们到现在，每次与他们的工作人员联系，他们平均能在半分钟之内给我提供服务，而且都特别有耐心，真的很不错。有几个网址我要推荐给大家，这也是我忙碌了好久从无数网站里找出的能帮你解决很多问题的网站。http://www.web-hosting-top.com 这个网站的最右边有个 Review，点击进入后在网页的下方有网友们给这个网站的打分，当时我选择了 Arvixe 很大一部分原因是看见网友对它的评价是最高的。http://www.webhostingtalk.com 这是一个关于hosting的一个论坛，在搜索框里输入你感兴趣的空间商的名字看看网友们怎么评价的，还有 http://www.hostsearch.com。最后给几个当时我关心过的空间商，性价比比较高的：http://www.2mhost.com, https://www.doteasy.com . 
  5. 谨防诈骗！一定要提高警惕，小心上当受骗，例如 www.m2host.com 就是个例子，它家给的东西超级便宜，但据说是个骗子，你在前面我给的论坛里搜索一下就明了了。

* 付款问题。最方便的付款方式是用visa或master的信用卡。付款后几秒钟就可以收到关于自己帐号信息的电子邮件。对于没有信用卡的朋友我建议先问一下周围有没有亲戚朋友有的，能借一下最好，要不然要稍微周折一点了。我当时就拜托我们系的一个前辈帮我用信用卡支付的，灰常方便。如果没信用卡，你可以选择用Paypal，现在Paypal已经能支持银联的大多数银行，你需要开通网上银行，然后去[Paypal](https://www.paypal.com/)首页注册一个帐号（帐号名字要与银行卡名字一样），把自己的银行卡绑定到Paypal上去,然后通过转账的方式付款，但据说交易是有延时的，就是你打上钱并不能立马收到邮件，要等一段时间才行。另外说个题外话：Paypal的服务不是一般好，我给他们发过一封电子邮件，然后过了一天他们给我打的国际电话帮我解决问题：）声援一下，哈哈。 最后说一个更简洁的办法：代购。找些信得过的代购帮你买，我这里推荐一下[美国主机侦查](http://bbs.idcspy.com/)，代购费仅十元人民币，论坛里的好友还是很给力的，有空可以去看看：）
